# Clinic Appointment System (ASP.NET MVC + ADO.NET + SQL Server)

## 📌 Project Overview
This project is a **Clinic Appointment Management System** developed using ASP.NET MVC, ADO.NET, and SQL Server.  
It allows patients to register, book appointments, and view appointment history. Doctors and administrators can manage schedules and track patient visits.

---

## ⚙️ Features
- Patient registration form with validation  
- Appointment booking and scheduling  
- Appointment history with patient and doctor details  
- SQL Server integration using ADO.NET  
- MVC architecture (Model, View, Controller)

---

## 🛠️ Technologies Used
- ASP.NET MVC  
- ADO.NET  
- SQL Server  
- Visual Studio 2022
